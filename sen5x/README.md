# Seeedstudio Grove - SEN54 All-in-one environmental sensor 

https://www.seeedstudio.com/Grove-All-in-one-Environmental-Sensor-SEN54-p-5374.html 

Measurements: VOC, RH, Temp, PM1.0/2.5/4/10 with proprietary algorithms, HVAC, Air Purifier

> NB: Power should be 5.0V

Install the [library](https://wiki.seeedstudio.com/Grove_SEN5X_All_in_One/) `Sensirion I2C SEN5X` by Sensirion through the `Tools > Manage Libraries ...`

https://github.com/Sensirion/arduino-i2c-sen5x/

Open, build and upload the sketch [`File > Examples > Sensirion I2C SEN5X > exampleUsage`](https://github.com/Sensirion/arduino-i2c-sen5x/blob/master/examples/exampleUsage/exampleUsage.ino)
